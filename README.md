# VEXORA 🛍️

A modern full-stack MERN e-commerce platform built with a premium UI/UX and real-world functionality.

---

## 🚀 Features

* 🔐 Authentication (JWT + Cookies)
* 🛒 Cart Management System
* ❤️ Wishlist / Save Product
* 🛍️ Product Listing & Filtering
* 👤 Seller Dashboard (Add / Manage Products)
* 📦 Order Flow Ready
* 💳 Razorpay Integration (in progress)
* 🌐 Fully Responsive UI

---

## 🛠️ Tech Stack

### Frontend

* React (Vite)
* Tailwind CSS
* Redux Toolkit
* Axios

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)
* JWT Authentication

---

## 📁 Project Structure

```
Snitch/
├── Frontend/
└── Backend/
```

---

## ⚙️ Environment Variables

### Backend (.env)

```
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
CLIENT_URL=http://localhost:5173
```

### Frontend (.env)

```
VITE_API_URL=http://localhost:3000
```

---

## ▶️ Run Locally

### Backend

```
cd Backend
npm install
npm start
```

### Frontend

```
cd Frontend
npm install
npm run dev
```
